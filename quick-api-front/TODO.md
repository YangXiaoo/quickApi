# 总体架构
## 测试端
> 只对本地启动的项目进行测试，不对外搜索，不需要登录
- 测试项目，左侧菜单
- 首页显示，提交历史
- 测试文档增删查改功能

## 用户端
- 项目接口文档查看
  - 项目搜索
- 用户自己保存的接口
  - 调用本地接口，调用本地接口时实际上是将localhost转换为本地IP，通过远程服务器调用本地IP
  - 调用远端接口，实际通过远程服务器将接口调用参数返回给前端
